Наименования 

Классы в Java собырают в пакеты (packages)
Пакеты обычно собирают в *.jar файлы (zip архивы)

Класс может быть однозначно идентиицирован по полному
имени packageName.drectoryName.ClassName

Имена пакетов и директорий принято начинать с маленькой буквы

Имена классов: MySuperClassName
Имена функций: getMyVariable()
Имена переменных: myVariable
Пакеты не могут иметь циклтческих зависимостей
